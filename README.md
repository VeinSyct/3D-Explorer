# 3D Product Explorer

A fully customizable **Three.js-based 3D Product Explorer** built for smooth, high-performance interactive experiences. Separate and reveal internal components, create dynamic camera transitions, and build stunning product visualizations or immersive virtual experiences.

Designed for flexibility and precision, this project empowers developers to:

* Showcase complex 3D products with fluid controls
* Add interactive features and camera paths
* Create exploded views and component breakdowns
* Adapt easily for product demos, educational visuals, engineering presentations, or mechanical inspections
* Deliver real-time, high-performance 3D experiences with Three.js

Transform static 3D assets into dynamic, interactive experiences with seamless precision control.

---

## Features

* Interactive 3D product exploration
* Smooth exploded-view animations
* Dynamic camera transitions
* Real-time component visibility control
* High-performance rendering with Three.js
* Fully customizable animation and interaction logic
* Compatible with custom GLTF/GLB assets
* Reusable virtual experience framework

## Getting Started

### Installation

```bash
git clone https://github.com/your-username/3d-product-explorer.git
cd 3d-product-explorer
npm install
```

### Run Locally

```bash
npm run dev
```

Open your browser:

```text
http://localhost:3000
```

---

## Create Your Own Product Experience

To build your own interactive product explorer:

1. Clone the project.
2. Replace the sample Electric Vehicle Power Drive Controller model with your own GLTF/GLB asset.
3. Configure component groups and exploded-view animations.
4. Customize camera positions, transitions, and interactions.
5. Deploy your experience.
